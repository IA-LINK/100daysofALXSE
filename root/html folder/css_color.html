Before beginning any troubleshooting steps on a customer's computer, a technician should follow these preparatory actions:

Ask the customer about the issue they are experiencing. ollect as much detail as possible about symptoms, error messages, recent changes, and any troubleshooting steps they may have already taken.

If the issue could potentially result in data loss or if troubleshooting involves making significant changes to the system, ensure that critical data is backed up.

Document the computer's hardware and software configuration, including specifications such as make, model, operating system version, installed applications, and any recent updates or changes.
